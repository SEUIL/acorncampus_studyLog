## 핵심 개발 규칙 (변경 없음)

- DB 쿼리: `PreparedStatement` 만 사용, `Statement` 금지 (SQL Injection 방지)

- DB 자원: 사용 후 반드시 `DBUtil.close(conn, pstmt, rs)` 호출
- AJAX 응답: Gson 사용 (`AdminController` 기준) — `String.format()` JSON 조립 금지
