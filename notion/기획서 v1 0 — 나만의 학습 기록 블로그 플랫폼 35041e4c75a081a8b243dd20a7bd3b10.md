# 기획서 v1.0 — 나만의 학습 기록 블로그 플랫폼

# 1. 프로젝트 개요

| 항목 | 내용 |
| --- | --- |
| 프로젝트명 | 나만의 학습 기록 블로그 플랫폼 |
| 목적 | 학습한 내용을 체계적으로 기록·관리하고, 성장 과정을 시각화하는 개인 기술 블로그 서비스 |
| 타겟 사용자 | 개발·학습 중인 개인 (주니어 개발자, 부트캠프 수강생 등) |
| 핵심 가치 | 단순 메모를 넘어 학습의 연속성과 성장을 시각화 |

---

# 2. 기술 스택

| 구분 | 기술 |
| --- | --- |
| Frontend | JSP, HTML/CSS, JavaScript (AJAX) |
| Backend | Java 17, Servlet 기반 MVC2 패턴 |
| Database | Oracle 19c |
| 인증 | 세션 기반 인증 (Session-based Authentication) |
| 에디터 | Toast UI Editor (마크다운 + 코드 하이라이팅) |
| 배포 | AWS (추후 확정) |

# 3. 주요 기능 목록

## 3-1. 회원 기능

### 회원가입

- 입력값: 이메일, 비밀번호, 닉네임
- 이메일 중복 검사 (AJAX 실시간)
- 비밀번호 규칙: 8자 이상, 영문 + 숫자 + 특수문자 포함
- 가입 완료 시 로그인 페이지로 리다이렉트

### 로그인 / 로그아웃

- 이메일 + 비밀번호 검증 후 HttpSession 발급
- 로그아웃 시 세션 무효화 (session.invalidate())
- 로그인 상태 유지: 세션 타임아웃 설정 (기본 30분)
- 로그인 전 접근 제한 페이지는 로그인 페이지로 리다이렉트

### 마이페이지

- 닉네임, 프로필 사진, 한줄 소개 수정
- 비밀번호 변경 (현재 비밀번호 확인 필요)
- 내가 쓴 글 목록 (공개/비공개 구분 표시)
- 회원 탈퇴 (소프트 삭제 — deleted_at 기록, 30일 후 완전 삭제)

---

## 3-2. 게시글 기능

### 게시글 작성

- 제목 (최대 100자)
- 본문: Toast UI Editor (마크다운)
    - 볼드, 이탤릭, 인용, 표, 코드 블록
    - 언어 선택 드롭다운 → 코드 하이라이팅
    - 이미지 드래그&드롭 업로드
- 태그 입력 (최대 5개, 쉼표 구분)
- 공개 / 비공개 토글
- 임시저장 (5분마다 자동저장, localStorage 활용)
- 썸네일 이미지 선택 (없으면 기본 이미지 자동 배정)

### 게시글 수정 / 삭제

- 본인 게시글만 수정·삭제 가능 (세션 기반 권한 체크)
- 삭제 시 확인 다이얼로그 표시
- 소프트 삭제 (deleted_at 컬럼 활용)

### 게시글 목록 조회

- 정렬: 최신순
- 필터: 태그 필터, 키워드 검색
- 페이지네이션 (한 페이지 10개)
- 비공개 글은 작성자 본인에게만 표시

### 게시글 상세 조회

- 마크다운 렌더링 + 코드 하이라이팅
- 조회수 카운트 (동일 세션 중복 카운트 방지)
- 이전 글 / 다음 글 네비게이션
- 관련 태그 클릭 → 해당 태그 게시글 목록 이동

### 좋아요 / 댓글

- 게시글 좋아요 / 싫어요 (1인 1회)
- 댓글 작성, 수정, 삭제
- 댓글 좋아요 / 싫어요

---

## 3-3. 태그 & 검색

- 태그 목록 페이지: 사용 빈도순 태그 클라우드 표시
- 태그 클릭 → 해당 태그 포함 게시글 목록 필터
- 키워드 검색: 제목 + 본문 검색
- 검색 결과 페이지에서 태그 필터 추가 적용 가능

---

## 3-4. 잔디 심기 (학습 기여도 시각화)

- 마이페이지 상단에 1년치 활동 캘린더 표시
- 날짜별 게시글 작성 수를 색상 농도(4단계)로 표현
    - 0개: 회색 | 1개: 연두 | 2개: 초록 | 3개↑: 진초록
- 칸 hover 시 날짜 + 작성 수 툴팁 표시
- 연속 작성 일수(스트릭) 표시

---

## 3-5. 시리즈 기능

- 게시글을 묶어 하나의 시리즈로 구성
- 시리즈 생성: 이름, 설명, 공개 여부
- 게시글 작성/수정 시 시리즈 선택 드롭다운
- 시리즈 내 글 순서 관리 (order 컬럼)
- 시리즈 페이지: 목차(TOC) 형태로 연재글 목록 표시
- 게시글 상세 하단에 "이 시리즈의 다른 글" 섹션 표시

---

## 3-6. AI 자동 태그 및 요약

- 게시글 본문을 LLM API로 분석
- 태그 자동 추천
- 3줄 내외 자동 요약

# 3-7. 신고 기능

- 게시글 / 댓글 신고 버튼 (로그인 사용자만 가능)
- 신고 사유 입력 (최대 500자)
- 동일 사용자가 동일 대상 중복 신고 불가 (애플리케이션 레이어 체크)
- 신고 접수 후 관리자 검토 대기 (PENDING 상태)

# 3-8. 관리자 기능

> 접근 조건: `role = 'ADMIN'` 인 사용자만 접근 가능. 일반 사용자 접근 시 403 또는 메인으로 리다이렉트.
> 

### 관리자 대시보드 `/admin`

- 주요 통계 한눈에 표시
    - 전체 회원 수 / 오늘 가입자 수
    - 전체 게시글 수 / 오늘 작성된 게시글 수
    - 처리 대기 중인 신고 수
    - 최근 7일 게시글 작성 현황 (간단한 수치)

### 회원 관리 `/admin/users`

- 전체 회원 목록 조회 (페이지네이션)
- 검색: 이메일 / 닉네임 키워드 검색
- 회원 상세 정보 확인 (가입일, 작성글 수, 탈퇴 여부)
- 강제 탈퇴 처리 (소프트 삭제, deleted_at 기록)
- 계정 정지 기능 (`is_banned` 컬럼으로 로그인 차단)

### 게시글 관리 `/admin/posts`

- 전체 게시글 조회 (비공개 글 포함)
- 검색: 제목 / 작성자 키워드 검색
- 관리자 강제 삭제 (소프트 삭제)
- 삭제 전 확인 다이얼로그 표시

### 댓글 관리 `/admin/comments`

- 전체 댓글 조회 (대댓글 포함)
- 검색: 내용 / 작성자 키워드 검색
- 관리자 강제 삭제 (소프트 삭제)

### 신고 관리 `/admin/reports`

- 신고 목록 조회 (상태별 필터: PENDING / RESOLVED / DISMISSED)
- 신고 상세 확인 (신고자, 대상 게시글/댓글, 신고 사유)
- 처리 액션:
    - **승인 (RESOLVED)**: 해당 게시글/댓글 강제 삭제 + 신고 처리 완료
    - **기각 (DISMISSED)**: 신고 내용 무효 처리
- 처리 일시 및 처리 관리자 ID 자동 기록

### 태그 관리 `/admin/tags`

- 전체 태그 목록 조회 (사용 빈도순)
- 태그 삭제 (연결된 post_tags 함께 삭제)
- 중복/유해 태그 정리 목적

# 4. 페이지 구성 (사이트맵)

- 전체 사이트맵 펼치기
    - **공통**
        - 헤더 (로고, 검색바, 로그인/마이페이지 버튼)
        - 푸터
    - **메인 페이지** `/`
        - 최신 게시글 목록
        - 인기 태그 클라우드
    - **회원**
        - 회원가입 `/join`
        - 로그인 `/login`
        - 마이페이지 `/mypage`
        - 회원정보 수정 `/mypage/edit`
    - **게시글**
        - 목록 `/posts`
        - 작성 `/posts/write`
        - 상세 `/posts/{postId}`
        - 수정 `/posts/{postId}/edit`
    - **태그**
        - 태그 목록 `/tags`
        - 태그별 게시글 `/tags/{tagName}`
    - **검색**
        - 검색 결과 `/search?q={keyword}`
    - **시리즈**
        - 시리즈 목록 `/series`
        - 시리즈 상세 `/series/{seriesId}`

# 5. 비기능 요구사항

| 항목 | 내용 |
| --- | --- |
| 보안 | XSS 방지 (출력 시 HTML 이스케이프), SQL Injection 방지 (PreparedStatement) |
| 세션 관리 | 세션 타임아웃 30분, 로그인 필요 페이지 접근 제어 |
| 소프트 삭제 | 게시글·회원 모두 deleted_at 컬럼으로 소프트 삭제 적용 |
| 파일 업로드 | 이미지 파일만 허용, 확장자 및 MIME 타입 검증, 최대 5MB |
| 페이지네이션 | 모든 목록 페이지 적용 (한 페이지 10개) |
