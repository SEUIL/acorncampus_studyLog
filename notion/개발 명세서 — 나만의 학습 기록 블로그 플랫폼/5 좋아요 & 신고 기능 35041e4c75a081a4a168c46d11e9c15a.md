# 5. 좋아요 & 신고 기능

# 좋아요/싫어요 기능명세서 (MVC패턴)

**Model / Service / DAO / DTO**

`LikeService`, `PostLikeDAO`, `CommentLikeDAO`, `LoginCheckFilter`

| 기능명 | 요청매핑 | 요청방식 | 서블릿 Controller | 응답데이터 | View | 재요청 (Redirect) |
| --- | --- | --- | --- | --- | --- | --- |
| 게시글 좋아요/싫어요 처리 | `/l_check/post/like.do` | POST | `PostLikeServlet` | 좋아요 수, 싫어요 수, 내 반응 JSON | - | - |
| 댓글 좋아요/싫어요 처리 | `/l_check/comment/like.do` | POST | `CommentLikeServlet` | 좋아요 수, 싫어요 수, 내 반응 JSON | - | - |

> 응답 형식 예시: `{ "status": "success", "likeCount": 12, "dislikeCount": 3, "myLike": "L" }`
> 

> 동일 반응 재클릭 시 취소(DELETE), 다른 반응 클릭 시 전환(UPDATE)
>

# 신고 기능명세서 (MVC패턴)

**Model / Service / DAO / DTO**

`ReportService`, `ReportDAO`, `ReportDTO`, `LoginCheckFilter`

| 기능명 | 요청매핑 | 요청방식 | 서블릿 Controller | 응답데이터 | View | 재요청 (Redirect) |
| --- | --- | --- | --- | --- | --- | --- |
| 게시글 신고 처리 | `/l_check/report/post.do` | POST | `ReportServlet` | 성공/실패/중복 JSON | - | - |
| 댓글 신고 처리 | `/l_check/report/comment.do` | POST | `ReportServlet` | 성공/실패/중복 JSON | - | - |

> 중복 신고 시: `{ "status": "error", "message": "이미 신고한 게시글입니다" }`
>
