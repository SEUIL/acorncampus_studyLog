# 3. 게시글 & 댓글 기능

# 게시글 기능명세서 (MVC패턴)

**Model / Service / DAO / DTO**

`PostService`, `PostDAO`, `PostDTO`, `LoginCheckFilter`

| 기능명 | 요청매핑 | 요청방식 | 서블릿 Controller | 응답데이터 | View | 재요청 (Redirect) |
| --- | --- | --- | --- | --- | --- | --- |
| 게시글 목록 조회 | `/post/list.do` | GET | `PostListServlet` | `ArrayList<PostDTO>` | `post_list.jsp` | - |
| 게시글 작성 페이지 이동 | `/l_check/post/write.do` | GET | `PostWriteServlet` | `ArrayList<SeriesDTO>` | `post_write.jsp` | - |
| 게시글 작성 처리 | `/post/write.do` | POST | `PostWriteServlet` | 실패 시 에러 코드 | - | 성공: `/post/detail.do?postId={id}` |
| 게시글 상세 조회 | `/post/detail.do` | GET | `PostDetailServlet` | `PostDTO`, `ArrayList<CommentDTO>`, 좋아요/싫어요 수 | `post_detail.jsp` | - |
| 게시글 수정 페이지 이동 | `/l_check/post/update.do` | GET | `PostUpdateServlet` | `PostDTO`, `ArrayList<SeriesDTO>` | `post_modify.jsp` | - |
| 게시글 수정 처리 | `/post/update.do` | POST | `PostUpdateServlet` | 실패 시 에러 코드 | - | 성공: `/post/detail.do?postId={id}` |
| 게시글 삭제 처리 | `/l_check/post/delete.do` | POST | `PostDeleteServlet` | 실패 시 에러 코드 | - | 성공: `/post/list.do` |
| 이미지 업로드 | `/post/upload.do` | POST | `ImageUploadServlet` | 이미지 경로 JSON | - | - |
