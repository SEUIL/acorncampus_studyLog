# 4. 태그 & 검색 & 시리즈

# 태그 기능명세서 (MVC패턴)

**Model / Service / DAO / DTO**

`TagService`, `TagDAO`, `TagDTO`

| 기능명 | 요청매핑 | 요청방식 | 서블릿 Controller | 응답데이터 | View | 재요청 (Redirect) |
| --- | --- | --- | --- | --- | --- | --- |
| 태그 목록 조회 (클라우드) | `/tag/list.do` | GET | `TagListServlet` | `ArrayList<TagDTO>` (사용 빈도 포함) | `tag_list.jsp` | - |
| 태그별 게시글 목록 | `/tag/post.do` | GET | `TagPostListServlet` | `ArrayList<PostDTO>`, 태그명 | `tag_post_list.jsp` | - |

# 시리즈 기능명세서 (MVC패턴)

**Model / Service / DAO / DTO**

`SeriesService`, `SeriesDAO`, `SeriesDTO`, `PostDAO`, `PostDTO`, `LoginCheckFilter`

| 기능명 | 요청매핑 | 요청방식 | 서블릿 Controller | 응답데이터 | View | 재요청 (Redirect) |
| --- | --- | --- | --- | --- | --- | --- |
| 시리즈 목록 조회 | `/series/list.do` | GET | `SeriesListServlet` | `ArrayList<SeriesDTO>` | `series_list.jsp` | - |
| 시리즈 상세 조회 | `/series/detail.do` | GET | `SeriesDetailServlet` | `SeriesDTO`, `ArrayList<PostDTO>` | `series_detail.jsp` | - |
| 시리즈 생성 페이지 이동 | `/l_check/series/write.do` | GET | `SeriesWriteServlet` | - | `series_write.jsp` | - |
| 시리즈 생성 처리 | `/series/write.do` | POST | `SeriesWriteServlet` | 실패 시 에러 코드 | - | 성공: `/series/list.do` |
| 시리즈 수정 페이지 이동 | `/l_check/series/update.do` | GET | `SeriesUpdateServlet` | `SeriesDTO` | `series_modify.jsp` | - |
| 시리즈 수정 처리 | `/series/update.do` | POST | `SeriesUpdateServlet` | 실패 시 에러 코드 | - | 성공: `/series/detail.do?seriesId={id}` |
| 시리즈 삭제 처리 | `/l_check/series/delete.do` | POST | `SeriesDeleteServlet` | 실패 시 에러 코드 | - | 성공: `/series/list.do` |
