# 1. 프로젝트 구조 & 공통 규칙

# Main/Header/Footer 기능명세서 (MVC패턴)

**Model / Service / DAO / DTO**

`EncodingFilter`, `LoginCheckFilter`, `DBUtil`, `BCryptUtil`

> ⚠️ **Java EE 8 환경** — 패키지명은 반드시 `javax.servlet.*` 사용. Java EE 9부터 `jakarta.*`로 변경되었으므로 혼용 절대 금지.
> 

## 의존성 (pom.xml)

| 라이브러리 | GroupId | ArtifactId | Version | scope |
| --- | --- | --- | --- | --- |
| Java EE 8 API | javax | javaee-api | 8.0 | provided |
| jBCrypt | org.mindrot | jbcrypt | 0.4 | compile |
| JSTL | javax.servlet | jstl | 1.2 | compile |
| Commons FileUpload | commons-fileupload | commons-fileupload | 1.5 | compile |
| Commons IO | commons-io | commons-io | 2.15.1 | compile |
| Oracle JDBC | [com.oracle](http://com.oracle).database.jdbc | ojdbc8 | 19.21.0.0 | compile |
| Toast UI Editor | CDN (unpkg) | - | 3.x | - |

> `javaee-api`는 `<scope>provided</scope>` 필수 — Tomcat이 이미 제공하므로 WAR에 포함하면 충돌 발생
>

## 패키지 구조

```java
src/
├── main/
│   ├── java/
│   │   └── com/blog/
│   │       ├── controller/     # Servlet (Controller 역할)
│   │       ├── service/        # 비즈니스 로직
│   │       ├── dao/            # DB 접근 (DAO)
│   │       ├── dto/            # 데이터 전달 객체
│   │       ├── filter/         # 서블릿 필터 (로그인 체크 등)
│   │       └── util/           # DBUtil, BCryptUtil 등
│   └── webapp/
│       ├── WEB-INF/
│       │   ├── views/          # JSP 파일
│       │   └── web.xml
│       └── resources/
│           ├── css/
│           ├── js/
│           └── upload/         # 업로드 이미지 저장
```

# 공통 필터 기능명세서 (MVC패턴)

**Model / Service / DAO / Filter**

`LoginCheckFilter`, `EncodingFilter`, `UserDto`

| 기능명 | 요청매핑 | 요청방식 | 서블릿 Controller | 응답데이터 | View | 재요청 (Redirect) |
| --- | --- | --- | --- | --- | --- | --- |
| 인코딩 설정 | `/*` | ALL | `EncodingFilter` | - | - | - |
| 로그인 체크 | `/posts/write`, `/mypage/*`, `/admin/*` 등 | ALL | `LoginCheckFilter` | - | - | `/login` (미로그인 시) |
| 관리자 권한 체크 | `/admin/*` | ALL | `LoginCheckFilter` | - | - | `/` (권한 없을 시) |
| 계정 정지 체크 | 로그인 시 | POST | `LoginCheckFilter` | - | - | `/login?banned=true` |
